=== Description ===
Pass Max is a multipurpose theme designed for flexible, easy site building. It includes a distraction-free template for page builders and is Gutenberg-optimised. RTL languages are supported, along with multiple header and footer options, customisable sitewide colour palettes, strong performance, WooCommerce enhancements, and companion plugin integration for prebuilt starter sites. The theme is lightweight, SEO-optimised, and verified for compatibility with popular tools and plugins. Demo: https://starter-sites.productiveminds.com/pass-max. Support: https://www.productiveminds.com/support
